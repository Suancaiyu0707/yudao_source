使用 IntelliJ IDEA 从 Fork 出来的仓库拉取代码，并导入。导入时，请记得下载 Standard library ，如下图所示 ：

导入完成后，Cargo 会自动下载 TiKV 项目的依赖库，可能下载时间较长，请耐心等待。此时适合，关注下笔者的微信公众号：【芋道源码】。
